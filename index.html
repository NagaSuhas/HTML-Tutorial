HTML Tutorial

"Roll Mouse over the tags to know more about it."

HTML TAG

This tag tells the browser that this is an HTML document and it represents the root of an HTML document.

HEAD TAG

This tag contains a title for the document, scripts, styles, meta information, and more.

BODY TAG

This tag contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

HEADER TAG

This element represents a container for introductory content or a set of navigational links.

IMAGE TAG

This tag defines an image in an HTML page and it has two required attributes: src and alt.

DIV TAG

This tag is often used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript.

TITLE TAG

This tag is required in all HTML documents and it defines the title of the document.

META TAG

This tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.

LINK TAG

This tag defines a link between a document and an external resource.

NAVIGATION TAG

This tag defines a set of navigation links.

UL TAG

This tag defines an unordered (bulleted) list and UL tag used together with the LI tag to create unordered lists.

BR TAG

This tag inserts a single line break.

BUTTON TAG

This tag defines a clickable button. Inside a BUTTON tag you can put content, like text or images.

LABEL TAG

This tag defines a label for a BUTTON, INPUT, METER, OUTPUT, PROGRESS, SELECT, or TEXTAREA element.

MAIN TAG

This tag specifies the main content of a document.

MARK TAG

This tag is used if you want to highlight parts of your text.

OPTION TAG

This tag defines an option in a select list.

PARAGRAPH TAG

This tag defines a paragraph.

PICTURE TAG

This tag gives web developers more flexibility in specifying image resources.

SPAN TAG

This tag is used to group inline-elements in a document and it provides no visual change by itself.

SAMP TAG

This tag is a phrase tag. It defines sample output from a computer program.

TABLE TAG

This tag defines an HTML table.

TEMPLATE TAG

This tag holds its content hidden from the client.

TR TAG

This tag defines a row in an HTML table.
